## What is Gacky-X?

**Gacky-X** is a professional-grade, prefix-based Discord security and moderation bot built to protect your server from modern threats. Designed with a **stateless architecture**, Gacky-X provides ultra-stable performance and OS-style interactive dashboards to make server management effortless.

Whether you are running a small community or a massive hub, Gacky-X guards your perimeter from raids, nukes, spam, and malicious links with precision.

## Key Features

*   **Antinuke Security ⭐**: 24/7 surveillance of staff actions with bitwise whitelist precision.
*   **Automod Protocols**: High-speed filters for spam, links, invites, caps, and emojis.
*   **Incident Reports**: Detailed audit trails for every security event with GX-ID tracking.
*   **Custom Bot Profile ⭐**: Per-guild bot customization including Nickname, Avatar, and Bio.
*   **Global Threat Intelligence**: Protection against known malicious users and phishing domains.
*   **Security Score**: Real-time evaluation of your server's security posture (0-100).
*   **Backup & Recovery**: Create snapshots and restore your server during emergencies.

<Note>
  **Note on Prefix:** Gacky-X uses prefix commands for stability and speed. The default prefix is **`-`**.
</Note>
