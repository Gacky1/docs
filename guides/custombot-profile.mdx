Stand out from other servers by giving Gacky-X a unique identity within your community.

## Per-Guild Identity (`-custombot`) ⭐

Most bots look the same in every server. Gacky-X allows you to change its appearance **without affecting other servers**.

### Customizable Fields:
- **Nickname:** Give the bot a server-specific name (e.g., "Gacky [STAFF]").
- **Avatar:** Upload a unique profile picture.
- **Banner:** Set a custom profile banner (requires Server Boosts for the guild).
- **Bio:** Add a custom description in the member list.

## How to Set a Custom Icon

1.  Type **`-custombot avatar`**.
2.  **Option A:** Paste a direct image URL in the modal.
3.  **Option B:** Attach an image file to the command: `-custombot avatar [upload-file]`.

## Technical Limits

- **File Size:** Max **8MB** per image.
- **Format:** Supports PNG, JPG, WEBP, and GIF.
- **Rate Limits:** You can update the profile once every **30 seconds** per server.

<Note>
  **Developer Logs:** Every profile change is logged to the bot owner's global developer channel to ensure transparency and prevent abuse.
</Note>
